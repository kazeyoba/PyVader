bool spaceInvader.game_run ( ty.List  alien_list,
ty.Dict  settings,
float  speed,
ty.Dict  spaceship_player 
)

The function will check the different conditions for continuing the game.

If the LIFE drops to 0, then True is returned. The player loses life when an alien reaches the value of the high_tray, moreover the game is reloaded.. If all aliens are destroyed, then a new list is generated, and the player's weapon is more powerful.

Parameters
spaceship_player(ty.Dict): the key shoot will allow to modify the power of the weapon.
speed(float): Accelerate or decelerate the speed of play.
alien_list(ty.List): The alien list that will be regenerated and or used in the game conditions.
settings(ty.Dict): Contains the high_tray which determines whether the alien has reached the end of the gameboard.
Returns
bool: If the key LIFE is 0, then it is game over, and returns TRUE.
